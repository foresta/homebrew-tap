# foresta/homebrew-tap

[Homebrew](https://brew.sh/) tap for [foresta](https://github.com/foresta) tools.

## Usage

```sh
brew tap foresta/tap
brew install foresta/tap/ctt
```

## Formulae

| Formula | Description | Upstream |
| --- | --- | --- |
| `ctt` | ClaudeCode Task management on Tmux | <https://github.com/foresta/ctt> |
